How to Use RoamKit

✈ Trips

  • Click + Add Trip in the sidebar to create a trip — give it a name, dates, and a color.
  • Click a trip to open it. Drag the ⠿ handle to reorder trips or move them between groups.
  • Hover a trip and click ⋮ to move, duplicate, export, or delete it.
  • Deleted trips land in Recently Deleted at the bottom of the sidebar — restore them anytime.

📋 Sections & Entries

  • Use + Add Section in the trip header to add sections. You can select multiple types at once — flights, lodging, activities, and more — all added in one go.
  • Each section type comes pre-built with relevant fields. Click + Add Entry to add a record.
  • Rename entries by clicking their title. Rename field labels by clicking them inline. Use + Field to add custom fields.
  • Hover a section header to rename it (icon + title), export it, import data into it, or delete it.
  • Use + Add Group to nest sections inside a group — handy for day-by-day itineraries.
  • Drag ⠿ to reorder sections or entries.

📤 Sharing

  • Every trip, section, and entry has a Share button that produces clean, formatted text you can copy and paste anywhere — email, texts, notes, wherever.
  • Sharing never uploads anything — it just formats your data for the clipboard.

📍 Destinations

  • Add a Destinations section to track the places you're visiting. Each one stores city, country, dates, and notes.
  • Expand Destination Resources on any destination for instant search links: Google, Maps, TripAdvisor, Viator, GetYourGuide, Yelp, Airbnb, and Hotels.
  • Use 🔍 Explore in the top bar (or welcome screen) to look up any place without needing to open a trip first.

↓ Export & Import

  • Sidebar — Export Trips / Import Trips backs up or restores your entire workspace. Import lets you choose Replace or Merge.
  • Trip header — export or import a single trip as a .json file.
  • Section header — export a section, or import a saved section file directly into it.

⚙ Data & Privacy

  • All data lives in your browser — nothing is sent to any server.
  • Use ⚠ Reset RoamKit at the bottom of the sidebar to wipe everything and start over (double confirmation required).

⌨ Open Source

RoamKit is open source — fork it, self-host it, or build your own version.
